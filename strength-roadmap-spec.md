# strength-roadmap — プロジェクト仕様書

## コンセプト

**「目標重量 × 達成日を入力 → 週ごとのマイルストーンを科学的根拠つきで逆算するダッシュボード」**

筋トレの目標を入力すると、経験レベルに基づいた現実的な週別プランを自動生成するシングルページアプリ。

## 入力パネル（左カラム）

- **種目タブ**：プリセット種目をバッジ形式で選択
  - ベンチプレス / スクワット / デッドリフト / オーバーヘッドプレス / バーベルロウ
- **現在の重量**（kg）：数値入力
- **目標重量**（kg）：数値入力
- **達成したい日**：カレンダーピッカー
- **「逆算する」ボタン**：クリックで結果エリアを更新

---

## 結果エリア（右カラム）

### メトリクスカード（3枚）
| カード | 内容 |
|---|---|
| 残り週数 | 今日から達成日までの週数 |
| 週あたり増量目安 | 逆算された1週間の増量kg |
| 達成難易度 | 経験レベルと伸び率から算出（余裕 / 標準 / 高め / 困難） |

### 進捗グラフ（Recharts LineChart）
- 横軸：週（Week 1, 2, 3...）
- 縦軸：重量（kg）
- **実線**：入力された目標に向かうライン
- **破線**：経験レベルから算出した現実的な推移ライン
- 2本を重ねることで目標の現実性を視覚化

### 週別マイルストーン テーブル
| カラム | 内容 |
|---|---|
| 週 | Week 1, 2, 3... |
| 目標重量 | その週に達成すべき重量（kg） |
| 推奨セット | 例：3×8、4×6 など |
| ステータス | 達成可能 / やや高め / 困難 |

## 逆算ロジック

科学的根拠をベースに計算。根拠の文言はUIには表示しない。

### 経験レベル別・週あたり最大伸び率

| レベル | 週あたり伸び率 |
|---|---|
| 初心者（〜6ヶ月） | 3〜5% |
| 中級者（6ヶ月〜2年） | 1〜2% |
| 上級者（2年以上） | 0.5〜1% |

根拠：過負荷原則（Progressive Overload）

> 実装メモ：上記は仕様書上の値。実装ではより現実的な数値（中級 ~0.75%、上級 ~0.2%）に下方修正している。詳細はソースコードコメント参照。

### マイルストーン計算の流れ

1. `totalWeeks` = 達成日までの週数を計算
2. `requiredGain` = 目標重量 - 現在重量
3. `weeklyTarget` = requiredGain / totalWeeks（理想的な線形増加）
4. `realisticWeeklyGain` = 現在重量 × 経験レベル別伸び率の中央値
5. 各週の `milestone` = 現在重量 + weeklyTarget × n（n = 週番号）
6. `difficulty` = weeklyTarget / realisticWeeklyGain の比率で判定
   - 〜0.7：余裕
   - 0.7〜1.0：標準
   - 1.0〜1.3：やや高め
   - 1.3〜：困難

### RM換算（推奨セット算出）

1RMを基準に各週の重量から推奨セット・レップ数を決定。
実装ではPrilepin's Chart（パワーリフティング世界標準）ベースに置き換え。

| 強度（1RM比） | セット×レップ |
|---|---|
| 〜70% | 4×10 |
| 70〜75% | 5×5 |
| 76〜85% | 5×3 |
| 86〜90% | 4×2 |
| 91%〜 | 3×1 |
